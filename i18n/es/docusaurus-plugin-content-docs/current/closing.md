# Cierre — Nombrando la Cosa

El oro nunca falló como activo.  
Falló como institución.

Mientras esta distinción permanezca sin reconocerse,  
el debate se repite — siempre con los mismos límites,  
siempre con las mismas fricciones.

Este documento no propone una conclusión.  
Simplemente nombra algo que siempre ha estado presente.

Y una vez nombrado,  
deja de ser invisible.
