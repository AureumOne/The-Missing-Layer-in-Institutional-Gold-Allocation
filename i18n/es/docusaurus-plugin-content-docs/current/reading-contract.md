# Contrato de Lectura

Este documento no es un argumento a favor del oro.  
No es una tesis de inversión.  
No es una propuesta de producto, estructura o discusión de retornos.

Este texto existe para un lector específico:  
alguien que ya comprende **por qué el oro tiene sentido**,  
y que también ha experimentado **por qué consistentemente falla en sostenerse institucionalmente**.

Si nunca has tenido que defender el oro ante un comité,  
si nunca te han exigido explicar responsabilidad, evidencia, límites operacionales o gobernanza,  
este documento probablemente no te será útil.

Nada aquí busca acuerdo.  
Nada aquí solicita una decisión.

Su único propósito es nombrar, con precisión técnica,  
una brecha estructural que siempre ha existido —  
pero que rara vez ha sido tratada como tal.
