# El Rol de Aureum (Deliberadamente Acotado)

Aureum existe únicamente para operar esta capa faltante.

Nada más.

No define estrategia de inversión.  
No mantiene activos de clientes.  
No actúa como custodio, corredor o intermediario financiero.

Su rol es estrictamente institucional:  
construir y mantener una infraestructura que permita que el oro físico  
sea integrado en entornos institucionales  
con gobernanza explícita, segregación funcional y evidencia verificable.

Aureum no resuelve el "caso del oro".  
Elimina la anomalía estructural que siempre ha impedido que el caso exista.
